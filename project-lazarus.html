PROJECT LAZARUS

Project Lazarus

First-Person Zombie Shooter | Unity | C# | Blender

G6078: Game Design and Development Assignment 3 | Grade: 92/100


Project Lazarus is a first-person, round-based zombie survival game inspired by Call of Duty: Zombies, developed in Unity with C#. I handled the full development cycle, including programming core gameplay mechanics and creating all 3D models and textures in Blender. This project demonstrates my skills in systems design, game development, and asset creation, delivering a cohesive and engaging gameplay experience.
