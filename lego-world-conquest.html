LEGO WORLD CONQUEST

Lego World Conquest

Digital Adaptation Of A Board Game | Unity | C# | Blender | Photoshop

G6046: Software Engineering | Grade: 67/100


Lego World Conquest is a digital adaptation of the classic board game Risk, reimagined with a Lego theme. Developed in Unity with C#, this was a university group project completed with four other developers based on a client’s specification. My primary responsibilities included creating all visual assets—from the main environment to UI elements—as well as implementing menu functionality and integrating UI interactions.
